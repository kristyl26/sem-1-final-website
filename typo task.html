Axial

Futura PT book oblique font is used. I used 13 pt font size for the description of
the event and 20 point font size for the date and the details of the event.

Radical

Description and the artist’s name divided and place line by line,
combining with the circle to form a sun. I used the type on a path function to do it.
Besides, Futura PT book oblique font is used.

Dilatational

Futura PT book oblique is used. Red color is used for specifying the date.
1 point stroke is used to specify the artist and “free”.

Random

Futura PT book oblique is used. Description is placed inside a tree letter by letter.
I had also smoothed the edge of the characters. Two rectangles are used while
the black one is used as a shadow to make the rectangle pop up.

Grid

Repeating rectangles and circles are used.
Clarendon URW light font is used.

Modular

Rectangles are used to form buildings and the windows.
The circle acts as the sun. Futura PT book oblique font is used.
Some of the texts are in 1 point stroke in order to draw the viewer’s attention.

Transitional

I had added rectangles and moved each point of
the rectangles to create the three mountains.
Description of the event are in lines with the flag
in the wrap effect to form the waves in a river.
Futura PT book oblique font is used.

Bilateral

Rectangles are used to form a border for the poster.
Futura PT book oblique font is used. The whole layout is simple.
1 point stroke is used to specify the date and the details of the event.
